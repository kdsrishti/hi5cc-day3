Loops

Loops allow handling repetitions easily

2 types

  • While
  • For

While loop

Be careful of noting the exit condition


							// while loop to draw 10 circles
							let posx = 0;
							function setup() {
							  createCanvas(400, 400);
							  background(230);
							}
							function draw() {
							  while (posx < 10) {
							    posx++;
							    fill(20);
							    circle(posx * 20, height/2, 10);
							  }
							}

Transformation

For transformation of the shapes in your sketch

Common types

  • Translate
  • Rotate


üìí Transform

Translate

Controls location displacement within the sketch. It is reset in every draw call.


							// translating on the canvas 
							function draw(){
							 background(110, 0, 0);
							 fill(0, 190, 255);
							 stroke(255);
							 circle(30, 30, 30);
							 translate(50, 50);
							 circle(30, 30, 30);
							 translate(50, 50);
							 circle(30, 30, 30);
							}

Rotate

Angles specified in Radians or Degrees. Positive numbers rotate shape clockwise.

						  rotate(PI/4);

Push

Push saves your current drawing styles. Used always with Pop


						 push(); // Start a new drawing state
						  strokeWeight(1);
						  fill(204, 153, 0);
						  translate(50, 0);
						  ellipse(0, 50, 33, 33);
						 pop(); // Restore original state

üìíPush

Pop

Pop removes the current drawing styles. Used always with Push


						 push(); // Start a new drawing state
